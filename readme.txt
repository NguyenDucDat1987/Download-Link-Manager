A powerful plugin to manage, track, and secure your download links with countdown timers and password protection.

== Description ==

Download Link Manager Pro is a simple yet powerful solution for managing file downloads on your WordPress site. 

**Key Features:**
*   **Link Masking:** Hide direct file paths to prevent hotlinking.
*   **Download Tracking:** Count exactly how many times your files are downloaded.
*   **Shortcode Generator:** User-friendly popup to insert download links into your content.
*   **Countdown Timer:** Force users to wait a few seconds before the download link appears.
*   **Password Protection:** Simple password barrier for your download pages.
*   **Ad Support:** Add advertising codes to your download page easily.
*   **Gutenberg & Classic Editor Support:** Works seamlessly with both editors.

== Installation ==

1.  Upload the `download-link-manager` folder to the `/wp-content/plugins/` directory.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  Go to **Downloads** in the admin menu to manage your files.
4.  Use the **"Insert Download Link"** button in your post editor to add links.

== Frequently Asked Questions ==

= How do I add a download link? =

= Can I link to external files? =
Yes, you can enter any URL (Google Drive, Dropbox, MediaFire, etc.) in the File URL field.

= Where can I see the download counts? =
You can see the total count in the "Downloads" list view or inside the edit screen of each download.
